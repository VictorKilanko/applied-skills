# Applied Skills

> Every skill. Applied.

AI-powered DIY repair assistant for cars, home appliances, plumbing, HVAC, and all vocational skills. Point your phone camera at the problem — get a diagnosis and step-by-step fix.

## Live Site

[victorkilanko.github.io/applied-skills](https://victorkilanko.github.io/applied-skills)

## What Is This?

Applied Skills is a consumer platform that closes the vocational skills gap using AI and computer vision. Starting with auto repair, expanding to every trade.

- Point your camera → AI diagnoses the problem
- Plug in OBD-II adapter → reads your car's computer directly
- Get exact repair steps + parts links

## Setup

To activate the waitlist form:
1. Sign up at [formspree.io](https://formspree.io)
2. Create a new form
3. Replace `YOUR_FORM_ID` in `index.html` with your Formspree form ID
